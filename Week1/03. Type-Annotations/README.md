## Definition

**Type Annotations:**

- Là 1 đoạn code mà ta dùng để nói với ts biến đó được tham chiếu đến loại giá trị nào.

**Type Inference:**

- TS sẽ cố tự động tìm ra loại giá trị nào mà 1 biến tham chiếu đên.

=> Nghe thì có vẻ giống như về định nghĩa thì đây là 2 định nghĩa hoàn toàn **ĐỐI LẬP**

**TA sẽ là dev dùng code để nói cho TS biết các kiểu mà ta đang làm việc** >< **TI chúng ta đang dựa vào TS để làm công việc đó cho ta**
